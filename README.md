# Swift Sword

> Adds a sword that grants a temporary speed boost when right-clicked.

Generated by [ModForge AI](https://lovable.dev) — Minecraft 1.21.11 (fabric).

## Build locally
```
./gradlew build
```
The compiled `.jar` will be in `build/libs/`.

## CI
GitHub Actions builds the `.jar` on every push. Download from the latest workflow run's artifacts.
